# 🚀 GameTour - Tournament & Live Score Platform

## 📌 Overview

**GameTour** is a mobile-first web application designed to manage sports tournaments, teams, and live scores in real-time.
It connects **organizers, players, and fans** on one platform for a seamless tournament experience.
## 🎯 Features

* 🏆 Tournament Creation & Management
* 👥 Team Registration System
* 📊 Real-Time Live Score Updates
* 🔔 Push Notifications (Location-based)
* 📸 Photo & Video Feed
* 👤 Player & Team Profiles
* 🧑‍💼 Organizer Dashboard

## 🛠️ Tech Stack

**Frontend:** HTML, CSS, JavaScript
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Real-time:** Socket.IO
**Cloud Storage:** AWS S3
**Notifications:** Firebase Cloud Messaging (FCM)
